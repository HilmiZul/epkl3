# e-PKL
Just for fun 😁

I build a simple app for manage PKL(Praktik Kerja Lapangan) data, like:
* company
* application letter
* guidance/mentoring

# Built with 💜
* Python 3.7.4
* Django 2.2.4
* mysqlclient 1.4.2.post1
* xlwt
* Materialize CSS
* Bootstrap 4.3.1
* Font Awesome 4.7.0
